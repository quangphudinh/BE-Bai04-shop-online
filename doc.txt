Các lệnh git:
git init
git status
git add .
git commit -m "noi dung"
git log
git show commit_id
git diff : show ra những dòng vừa thêm hoặc chỉnh sửa
gitk : show dashboard chi tiết
git checkout -- filename : quay lại file trước khi sửa
git reset HEAD ten_file HOẶC git reset ten_file : đẩy file từ vùng Staging về vùng Working
git reset -- soft commit_id : chuyển trạng thái từ Repository về lại Staging
git reset --mixed commit_id : chuyển từ repository về working

Nhánh trong GIT :
git branch : xem các danh sách branch
git checkout -b name_branch : tạo một nhánh mới và chuyển sang nhánh đó
git checkout ten_nhanh : chuyển sang nhánh khác
git merge ten_nhanh : để merge nhánh khác vào trong nhánh hiện tại
git branch -D tên_nhánh : xóa nhánh đó
